# 🚀 Быстрый старт P2P тестирования

## 📋 Что нужно для тестирования

- **Два компьютера** в одной WiFi сети
- **Go 1.21+** установлен на обоих
- **ProjectT** склонирован на оба устройства

---

## 🎯 Вариант 1: Быстрое тестирование (рекомендуется)

### Шаг 1: Запуск на ПК 1

```bash
cd c:\Users\egors\Desktop\projectT
go run cmd/p2p_diag/main.go --mode=host --port=4001 --dht=true --advertise
```

Запомните адрес из консоли:
```
📋 Сообщите этот адрес клиенту:
   projectt://QmABC123...@/ip4/192.168.1.5/tcp/4001/p2p/QmABC123...
```

### Шаг 2: Запуск на ПК 2

```bash
cd c:\Users\egors\Desktop\projectT
go run cmd/p2p_diag/main.go --mode=client --port=4001 --dht=true --target="projectt://QmABC123...@/ip4/192.168.1.5/tcp/4001/p2p/QmABC123..."
```

### Шаг 3: Проверка подключения

На любом ПК введите:
```
/peers
```

Должно показать:
```
📊 Подключённые пиры: 1
   • QmDEF456... [connected]
```

### Шаг 4: Тестирование связи

На ПК 1:
```
Привет, ПК 2!
```

На ПК 2:
```
Привет, ПК 1!
```

Сообщения должны приходить с логом:
```
[15:30:45.123] 💬 Сообщение: Привет, ПК 2!
```

## 🔍 Диагностика проблем

### Проблема: Таймаут подключения

**Лог:**
```
❌ ОШИБКА ПОДКЛЮЧЕНИЯ (30.00 сек): context deadline exceeded
```

**Причина:** Брандмауэр блокирует порт

**Решение:**
```bash
netsh advfirewall firewall add rule name="ProjectT P2P" dir=in action=allow protocol=TCP localport=4001
```

### Проблема: Connection refused

**Лог:**
```
❌ ОШИБКА ПОДКЛЮЧЕНИЯ (0.05 сек): connection refused
```

**Причина:** Сервер не запущен или порт закрыт

**Решение:**
1. Убедитесь что ПК 1 запущен в режиме `host`
2. Проверьте что порт 4001 не занят: `netstat -an | findstr 4001`

### Проблема: DHT не находит пиры

**Лог:**
```
📊 Поиск завершён (30.00 сек) - найдено пиров: 0
```

**Причина:** DHT требует времени на bootstrap

**Решение:**
1. Подождите 30-60 секунд
2. Используйте прямое подключение вместо DHT
3. Добавьте bootstrap пиры

## 📊 Команды для тестирования

| Команда | Описание |
|---------|----------|
| `/peers` | Показать подключённые пиры |
| `/connect <addr>` | Подключиться к пиру |
| `/advertise` | Рекламировать себя в DHT |
| `/find` | Найти пиры через DHT |
| `/info` | Показать информацию об узле |
| `/quit` | Выход |
